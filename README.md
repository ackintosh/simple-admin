#Simple Admin
Sinatraの練習で作った、シンプルな管理画面です。  
![simple admin](https://dl.dropbox.com/u/22083548/github/simple-admin/simple_admin.png)  

同時編集を警告する機能があります。  
![conflict_check](https://dl.dropbox.com/u/22083548/github/simple-admin/conflict_check.png)

##setup
    $ git clone
    $ cd simple-admin
    $ bundle install
    $ rackup
